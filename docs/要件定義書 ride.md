## **1\. システム概要**

「**ride**」は、バイク・車オーナーのためのAIライフログ・プラットフォームです。 愛車のコンディション管理（Health）、限られた時間での最適な走行体験（Explore）、そしてその記録を映画のように昇華させる制作支援（Create）を、AIによって一気通貫で提供します。

## **2\. ターゲットユーザー**

* **多忙なパパ・ママライダー**: 育児や仕事の隙間時間で効率的に趣味を楽しみたい方。  
* **シネマティック・クリエイター**: SNS（Instagram等）でハイクオリティな動画・画像発信を行いたい方。  
* **こだわりを持つオーナー**: 愛車（Zephyr χ等）の状態を正確に把握し、長く大切に乗り続けたい方。  
* **色彩表現を追求する方**: 色覚特性をAIで補完し、客観的に美しい色彩美を実現したい方。

---

## **3\. 主要機能要件**

### **3.1 【Health】AIコンディション診断**

愛車の「健康」を可視化し、維持管理の不安を解消します。

* **エンジン異音検知**:  
  * 録音されたエンジン音を過去の正常データと比較。  
  * 異常（タペット音、タッキング音等）の予兆をスコアリング。  
* **消耗品ビジュアル診断**:  
  * タイヤ、チェーン、プラグ等の写真を解析し、摩耗・劣化状況を判定。  
* **スマート・メンテナンスログ**:  
  * 入力された走行距離に基づき、オイル交換や車検のタイミングを算出してリマインド。

### **3.2 【Explore】インテリジェント・ルート提案**

「今、この瞬間」に最も価値のあるライディングルートを提案します。

* **タイムマネジメント・ルート検索**:  
  * 「帰宅まで○分」という制約内で、最適な走行ルートを算出。  
* **フォトジェニック・スポット予測**:  
  * 天候、太陽の角度、季節性を加味し、最も「映える」撮影地点を提案。  
* **ライディング・ガイド**:  
  * 目的地での最適な撮影アングルや、安全走行のためのリアルタイムな路面状況フィードバック。

### **3.3 【Create】シネマティック・エディター**

外部ソフト（DaVinci Resolve等）を使わず、スマホ一つで作品を完結させます。

* **AI Image-to-Image グレーディング**:  
  * 車両の造形を維持したまま、指定した映画風の色調（ルック）へ変換。  
* **カラー・ノーマライザー（客観補正）**:  
  * 波形解析に基づき、AIが「技術的に正しい色味」へ自動修正。

## **4\. 技術スタック・開発環境**

* **フロントエンド**: React 18 + Vite
* **バックエンド**: Node.js + TypeScript + Express
* **AIモデル**:  
  * **マルチモーダル解析**: Gemini 1.5 Pro / 3.0  
  * **画像加工**: Stable Diffusion (ControlNet併用)  
* **インフラ**: Google Cloud Run, Supabase  
* **開発ツール**: VSCode, Gemini CLI, codex, Docker Desktop, GitHub Desktop

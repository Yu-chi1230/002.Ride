# カスタムルール

※ ここに私が思考・行動する際に必ず守ってほしいルールや手順を記載してください。
（例：必ず丁寧語を使うこと、コンポーネントは関数型コンポーネントのみを使用すること、など）
・npmコマンドを使ってパッケージ等をインストールする時、または提案するときには、事前にそのパッケージが存在することを確認し、報告してください。私が許可した場合にインストールを実行してください。（幻覚スクワッティング対策）

・パッケージ等をインストールする際は、必ずコンテナ内にインストールしてください。
　コンテナがわからない場合は私に聞いてください。

・ハルシネーション（嘘）を起こしていると少しでも感じたら私に報告してください。

・指示に対しては、まず計画のみを報告してください。
　計画の報告時は、実行するかしないか、それによるメリットとデメリットがあれば提示をすること。

・Implementation plan は日本語で記載すること。
